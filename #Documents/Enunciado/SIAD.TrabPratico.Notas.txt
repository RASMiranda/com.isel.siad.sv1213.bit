Os dados necessários para desenvolver o trabalho prático estão disponíveis no servidor SQL2008 siad.e.ipl.pt:40022, na Base de dados BIT.

Para se autenticarem, utilizem SQL authentication, com as seguintes credenciais:
login: SIAD_DWCREATOR
password: SIAD_DWCREATOR

Todo o processamento de ETL não deve ser feito directamente sobre o referido servidor. Devem fazer uma cópia temporária para a vossa máquina de trabalho, sobre a qual será efectuado o processamento. 
Diariamente são adicionados novos registos.
A folha de cálculo que contém informações adicionais sobre os custos da empresa pode ser descarregada daqui: http://siad.e.ipl.pt/custos.xls

É também disponibilizada uma máquina virtual que contém todo o software utilizado na cadeira. Esta pode ser descarregada de 
\\ssicsrv3.ssic.deetc.win.isel.ipl.pt\vm_siad, apenas dentro da rede do ISEL.
Se forem pedidas credenciais de acesso usem o login isel-deetc-alun\alunoxp sem password. Para executarem a máquina virtual devem utilizar o software VirtualBox. 

As credenciais da máquina virtual são:
login: aluno_siad
password: _siadpwd
